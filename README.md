# 💰 Previsão de Renda com Machine Learning

Projeto final desenvolvido em **Python no Google Colab** para prever faixas salariais utilizando o *Adult Income Dataset*. 

O objetivo principal foi classificar se uma pessoa ganha mais ou menos de US$ 50.000 por ano, focando na otimização da métrica **F1-Score** devido ao desbalanceamento dos dados.

---

## 🏆 Resultados
* **Algoritmo:** Gradient Boosting Classifier
* **Performance Máxima (Classe Alvo >50K):** `0.7235` de F1-Score

---

## 🛠️ Tecnologias Utilizadas
* **Ambiente:** Google Colab
* **Linguagem:** Python
* **Bibliotecas:** Pandas, NumPy, Scikit-Learn, Matplotlib e Seaborn

---

## 📋 Etapas do Projeto
1. **Análise dos Dados:** Exploração do dataset e identificação do desbalanceamento de classes.
2. **Pré-processamento:** Limpeza de dados e codificação de variáveis categóricas.
3. **Treinamento:** Ajuste do modelo Gradient Boosting.
4. **Avaliação:** Foco total nas métricas de Precision, Recall e F1-Score.
